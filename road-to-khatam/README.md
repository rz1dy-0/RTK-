# Sistem Road To Khatam Al-Quran

Fail yang perlu upload ke GitHub:

1. index.html
2. style.css
3. script.js

Versi ini tidak perlukan database. Data disimpan dalam browser menggunakan localStorage.

Cara publish:
1. Buka GitHub.
2. Klik New repository.
3. Nama repository: road-to-khatam
4. Pilih Public.
5. Klik Create repository.
6. Klik Add file > Upload files.
7. Upload index.html, style.css dan script.js.
8. Klik Commit changes.
9. Pergi Settings > Pages.
10. Source pilih Deploy from a branch.
11. Branch pilih main.
12. Folder pilih /root.
13. Klik Save.
14. Tunggu link GitHub Pages keluar.

Nota: Data hanya kekal pada browser yang sama. Kalau mahu data dikongsi antara semua device, perlu sambung database seperti Firebase atau Supabase.
